# 文件夹功能使用示例

## 场景 1：创建工作相关的文件夹

假设你有以下图标：
- Gmail
- Google Drive
- Slack
- Notion

**操作步骤：**
1. 拖动 Gmail 图标到 Google Drive 图标上
2. 系统自动创建一个"新文件夹"，包含这两个图标
3. 右键点击文件夹，选择"重命名"，改名为"工作"
4. 拖动 Slack 图标到"工作"文件夹上
5. 拖动 Notion 图标到"工作"文件夹上

**结果：**
现在你有一个名为"工作"的文件夹，包含 4 个工作相关的应用。

## 场景 2：整理社交媒体

假设你有以下图标：
- Twitter
- Facebook
- Instagram
- LinkedIn

**操作步骤：**
1. 拖动 Twitter 到 Facebook 上，创建文件夹
2. 重命名为"社交媒体"
3. 将 Instagram 和 LinkedIn 拖入文件夹

**使用：**
- 点击"社交媒体"文件夹
- 在弹出的对话框中点击任意图标即可打开对应网站

## 场景 3：从文件夹中移除图标

假设你不再需要 LinkedIn 在"社交媒体"文件夹中：

**操作步骤：**
1. 点击"社交媒体"文件夹打开
2. 将鼠标悬停在 LinkedIn 图标上
3. 点击右上角出现的 ❌ 按钮
4. LinkedIn 图标会回到网格中

## 场景 4：文件夹自动解散

假设"社交媒体"文件夹只剩下 Twitter 和 Facebook：

**操作步骤：**
1. 打开文件夹
2. 移除 Twitter
3. 文件夹自动解散，Facebook 回到网格中

## 场景 5：跨页面添加到文件夹

假设你在第 1 页有一个"开发工具"文件夹，在第 2 页有 GitHub 图标：

**操作步骤：**
1. 在第 2 页拖动 GitHub 图标
2. 将它拖到侧边栏的第 1 页按钮上（等待页面切换）
3. 继续拖动到"开发工具"文件夹上
4. GitHub 图标会从第 2 页移动到第 1 页的文件夹中

## 提示和技巧

1. **快速整理**：一次性创建多个文件夹来整理不同类别的网站
2. **灵活调整**：随时可以从文件夹中移除或添加图标
3. **跨页面管理**：文件夹可以在不同页面之间移动
4. **拖拽排序**：文件夹可以像普通图标一样排序
5. **右键菜单**：使用右键菜单快速重命名或删除文件夹
